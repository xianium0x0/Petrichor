# images フォルダ

このフォルダにサイトで使用する画像を格納します。

## メンバー写真の差し替え方法

写真は `data/members.json` の各メンバーの `photo`（表面）・`photoBack`（裏面）フィールドに
`images/` フォルダ内のファイルパスを指定してください。

```json
{
  "name": "真夜 詩音",
  "photo":     "images/mayo-front.jpg",
  "photoBack": "images/mayo-back.jpg",
  ...
}
```

- `photo` — カード表面のシルエット背景として表示される写真
- `photoBack` — カードクリック時に開くモーダルのメイン画像として表示される写真
- どちらも `null` のままにするとイニシャルが表示されます
- `photoBack` が `null` の場合、`photo` がモーダルにも使用されます

## 推奨ファイル名・サイズ

| 用途 | 例 | 推奨サイズ | 備考 |
|---|---|---|---|
| メンバーカード表面（シルエット背景） | `mayo-front.jpg` | 600×800px（縦長）推奨 | カードの正方形枠に中央トリミングされる |
| メンバーモーダル画像 | `mayo-back.jpg` | 600×800px（3:4比率）推奨 | 縦長モーダルに全身が表示される。省略時は `photo` を使用 |
| ヒーロー背景（PC） | `hero-bg.jpg` | 1920×1080px | — |
| ヒーロー背景（スマホ） | `hero-bg-sp.jpg` | 768×1334px | なければPC用にフォールバック |
| OGP画像 | `og-image.jpg` | 1200×630px | — |
| ファビコン | `favicon.ico` | 32×32px | — |
